Pugsby's hyprland dotfiles<br>
====================================================<br>
Some commands may use sudo, you will have to use your password around 5 times total<br>
Please note that these dotfiles are in beta and installation may not succeed.
# Keybinds
- Super + Return     Kitty/Terminal
- Ctrl+Alt+T         Kitty/Terminal
- Super + F11          Toggle Float
- Super + L                    Lock
- Super + E     File Manager / Nemo
- Print Screen           Screenshot
- Alt+F4       Close current window
- Super           App Menu / Fuzzel
- Super+Arrow        Move Selection
- Super+[Number]        Change Desk
- Super+Scroll          Change Desk
# Installation
```
git clone https://github.com/pugsby/hyprlandDots
cd hyprlandDots
chmod +x install.sh
./install.sh
```
